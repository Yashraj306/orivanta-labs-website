# Orivanta Labs

A premium agency website for Orivanta Labs — a digital growth agency that builds websites, AI automation, and local SEO systems for restaurants and local businesses.

## Tech Stack

- **Framework**: TanStack Start (React + Vite)
- **Styling**: Tailwind CSS v4
- **Forms**: Netlify Forms
- **Deployment**: Netlify

## Pages

| Route | Description |
|---|---|
| `/` | Home — Hero, Services overview, How It Works, Before & After, Why Us, Testimonials, Contact CTA |
| `/services` | Full breakdown of all 6 services with feature lists |
| `/portfolio` | Featured project (Ajora Ethiopian Kitchen) + 3 additional case studies |
| `/about` | Company story, values, and team |
| `/contact` | Full contact form with free audit request |

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

For Netlify features (forms, etc.), use the Netlify CLI:

```bash
netlify dev --port 8889
```

## Design System

- **Primary**: `#004B49` (dark teal-green)
- **Accent**: `#00D4C8` (bright cyan)
- **Background**: White + `#F8F6F1` (warm cream)
- **Typography**: Inter (Google Fonts)
